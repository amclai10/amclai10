Project Overview

Noah McLain's Car Detailing Co.

Project Keypoints

  • Application description & purpose

    Noah McLain’s Car Detailing is a website that provides information on a car detailing service. It will go in detail on how it operates, what services are provided, and ways to book an appointment.

  • Target users

    Locals in need of a car detailing service.

  • Content overview

    This site will have forums for opening a support ticket and booking an appointment. It will also include a breakdown of the services provided and their costs. A page for customer testimonials will also be included.

Site Map

Page Design

  • Home Page

    • Purpose:

      This page is the landing page, the first thing people will see when they go to the website. It will include a navigation bar to access each subpage.

    • Target Audience:

      The target audience of this page will be customers.

    • Page Content:

      The home page will have all the subpages embedded within it, using JS to dynamically load each of them.

  • About Us

    • Purpose:

      The about us page will be used to learn about the company and its vision. It will include biographical information on the owner and how he operates.

    • Target Audience:

      The target audience of this page will be customers.

    • Page Content:

      This page will include an image of the company’s owner.

  • Frequently Asked Questions (FAQ)

    • Purpose:

      The FAQ page will be used to answer the most asked questions.

    • Target Audience:

      The target audience of this page will be customers.

    • Page Content:

      This page will also include a link to the contact page if the customer’s question was not listed.

  • Testimonials

    • Purpose:

      This page will provide testimonials from preexisting clients, asserting the quality of the services.

    • Target Audience:

      The target audience of this page will be customers.

    • Page Content:

      This page will include reviews from the customers, which will have testimonials, an image of the finished product, and a rating.

  • Contact

    • Purpose:

      This page will be used to provide contact details to interested clients.

    • Target Audience:

      The target audience of this page will be customers.

    • Page Content:

      This page will include a forum to submit a service inquiry, asking for the customer’s contact info and the related question to open a ticket. It will also include general contact information, such as an email and phone number.

  • Services

    • Purpose:

      This page will be used to showcase what services are offered.

    • Target Audience:

      The target audience of this page will be customers.

    • Page Content:

      It will include info on what is provided in each package, a forum to make an appointment, and images that visualize the finished products. The appointment forum will ask for the customer’s contact info, a date, and information on the customers car.
